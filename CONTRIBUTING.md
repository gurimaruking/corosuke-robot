# Contributing to Corosuke Robot / コロ助ロボットへの貢献

[English](#english) | [日本語](#japanese)

---

<a name="english"></a>
## English

Thank you for your interest in contributing to the Corosuke Robot project!

### How to Contribute

1. **Fork the repository**
2. **Create a feature branch** (`git checkout -b feature/amazing-feature`)
3. **Commit your changes** (`git commit -m 'Add amazing feature'`)
4. **Push to the branch** (`git push origin feature/amazing-feature`)
5. **Open a Pull Request**

### Areas for Contribution

- **Firmware**: Bug fixes, new features, optimization
- **3D Models**: Improved designs, new parts, print optimization
- **Documentation**: Translations, tutorials, assembly guides
- **Server**: New integrations, performance improvements

### Code Style

- **C++**: Follow Arduino/ESP32 conventions
- **Python**: Follow PEP 8
- **OpenSCAD**: Use descriptive variable names, comment complex logic

### Issues

Feel free to open issues for:
- Bug reports
- Feature requests
- Questions about the project

<a name="japanese"></a>
## 日本語

コロ助ロボットプロジェクトに興味を持っていただきありがとうございます！

### 貢献の方法

1. **リポジトリをフォーク**
2. **機能ブランチを作成** (`git checkout -b feature/amazing-feature`)
3. **変更をコミット** (`git commit -m '素晴らしい機能を追加'`)
4. **ブランチにプッシュ** (`git push origin feature/amazing-feature`)
5. **プルリクエストを作成**

### 貢献できる領域

- **ファームウェア**: バグ修正、新機能、最適化
- **3Dモデル**: デザイン改善、新パーツ、印刷最適化
- **ドキュメント**: 翻訳、チュートリアル、組立ガイド
- **サーバー**: 新しい統合、パフォーマンス改善

### コードスタイル

- **C++**: Arduino/ESP32の規約に従う
- **Python**: PEP 8に従う
- **OpenSCAD**: わかりやすい変数名、複雑なロジックにはコメント

### Issue

以下の場合はIssueを作成してください：
- バグ報告
- 機能リクエスト
- プロジェクトに関する質問
